# AE2 Pattern Highlighter

**AE2 Pattern Highlighter** is a client-side addon for Applied Energistics 2 (AE2) that helps players locate pattern providers directly from the Crafting CPU interface.

![Highlight Effect](https://github.com/YourUsername/ae2-pattern-highlighter/raw/main/screenshot.png)

## âœ¨ Features

- **Quick Locator**: Adds a `?` button to each crafting task in the Crafting CPU monitor.
- **Visual Highlighting**: Highlights the target Pattern Provider in the world with a rainbow X-ray outline visible through walls.
- **Provider Info**: Displays the coordinates in chat with clickable teleport commands.
- **Smart Observation**: Adds a `ğŸ‘` button to enter "Free Camera Mode" which automatically flies your view to the target block and orbits around it.
- **Multiplayer Safe**: Highlighting and camera effects are client-side only and do not affect other players.
